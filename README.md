# 🗂️ Advanced Skills Analytics

سكربت بايثون احترافي لتحليل البيانات المتداخلة (Nested Dictionaries) واستعراض مهارات المطورين ونسب إنجازهم بدقة عالية.

---

### 🚀 المميزات (Key Features)
* 🧠 **Deep Mapping:** التعامل مع هيكلة البيانات المعقدة (Nested Structures).
* 🔄 **Optimized Iteration:** استخدام تقنية `.items()` للوصول المباشر للبيانات (Efficient & Clean).
* ⚡ **Formatted Output:** عرض المخرجات بشكل شجري (Tree-style) يسهل قراءته.

### 🛠️ التقنيات المستخدمة (Tech Stack)
* **Language:** Python 3.x
* **Method:** Dictionary Unpacking & Nested Looping.
